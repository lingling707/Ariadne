[edit] 기본 도움말

게임을 진행하려면 마우스 좌클릭 또는 스페이스 바, 엔터 키를 누르세요. 선택지 화면에서는 좌클릭으로 선택지를 고르거나, 화살표 키로 선택지를 고르고엔터 키로 선택할 수 있습니다..

[edit] 게임 메뉴

게임을 플레이하는 도중, 마우스 좌클릭 또는 Esc 키를 입력하면 게임 메뉴를 열 수 있습니다. 게임 메뉴에서 선택할 수 있는 기능은 다음과 같습니다.

돌아가기
게임 화면으로 돌아갑니다.
저장하기
세이브 슬롯을 클릭해 게임을 저장할 수 있습니다.
불러오기
세이브 슬롯을 클릭해 게임을 불러올 수 있습니다. "자동" 버튼을 클릭하면 자동 저장 슬롯을 확인합니다.
환경설정
게임 환경을 설정합니다.:
화면 모드
창 모드나 전체화면 모드로 변경합니다.
언어
게임 화면에서 표시할 언어를 선택합니다.
텍스트 표시 속도
텍스트 표시 속도를 제어합니다. 슬라이더를 오른쪽으로 이동할 수록 텍스트 표시 속도가 빨라집니다. 맨 오른쪽으로 이동하면 텍스트가 한번에 표시됩니다.
조이스틱
조이스틱을 사용해 게임을 진행할 수 있도록 합니다.
스킵
게임을 진행하면서 한 번 확인한 메시지를 스킵하거나, 모든 메시지를 스킵하도록 선택합니다.
스킵 시작
대사를 스킵하기 시작하고 게임 화면으로 돌아갑니다.
선택지 이후
선택지에 도달했을 때 스킵을 멈출 것인지 말 것인지 선택합니다.
자동 진행 시간
자동 진행 시간을 조절합니다. 슬라이더가 왼쪽으로 이동할 수록 다음으로 넘어가기까지 필요한 시간이 짧아집니다. 맨 오른쪽으로 이동하면 자동 진행을 하지 않습니다.
배경음, 효과음, 음성 크기
배경음, 효과음, 음성 채널의 크기를 조절합니다. 슬라이더가 오른쪽으로 이동할 수록 소리 크기가 커집니다.
메인 메뉴
현재 게임을 종료하고 메인 메뉴로 돌아갑니다.
도움말
현재 보고 있는 도움말 화면을 표시합니다.
나가기
게임을 종료합니다.

[edit] 키와 마우스 버튼의 기능

좌클릭, Enter 키
게임을 진행하고, 선택지, 버튼, 슬라이더를 선택합니다.
Space 키
게임을 진행하지만 선택지를 선택하지는 않습니다.
화살표 키
선택지, 버튼, 슬라이더를 고릅니다.
Ctrl키
Ctrl 키를 누르고 있는 동안 메시지를 스킵합니다.
Tab 키
Tab 키를 누를 때마다 스킵 기능을 시작하거나 중단합니다.
마우스 휠 올리기, PageUp 키
롤백합니다. 롤백은 게임을 이전 상태로 돌리는 것으로, 이전 텍스트를 표시하거나 선택지를 변경할 수 있습니다.
마우스 휠 내리기, PageDown 키
롤포워드를 실행해 이전 롤백을 취소합니다.
좌클릭, Esc 키
게임 메뉴를 엽니다. 게임 메뉴가 열려있을 때에는 게임으로 돌아갑니다.
휠 버튼 클릭, H 키
텍스트 창이나 기타 UI를 숨깁니다.
F
전체 화면 모드나 창 모드로 전환합니다.
S
스크린샷을 찍어 screenshotxxxx.png 파일에 저장합니다. 파일 이름의 xxxx는 스크린샷을 찍을 수록 증가하는 숫자입니다.
Alt-M, Command-H
창을 작업표시줄로 이동시킵니다.
Alt-F4, Command-Q
게임을 종료합니다.
Delete 키
선택 중인 세이브 슬롯을 삭제합니다.
